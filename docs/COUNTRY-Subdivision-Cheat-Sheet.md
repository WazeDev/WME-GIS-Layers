**Tips:**
- Always use UPPERCASE for `COUNTRY` and `subL1` codes.
- Always use Proper Case for `subL2`
- For country-wide layers, use the country code in both the `COUNTRY` and `subL1`.
- For USA territories, use `PR` (Puerto Rico), `VI` (Virgin Islands), etc., in `subL1`.
